After reading this document, you should be able to understand what is the network capture in JazzVPN

## What do we capture?

Network auditory is a feature in JazzVPN to ensure that server administrators will always be able to trace user actions in the case they want it. Network auditory can be enabled using policies or on some event callbacks.
